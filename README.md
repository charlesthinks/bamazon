### Clone

- Clone this repo to your local machine using `https://github.com/charlesthinks/bamazon.git`

### Setup

> update and install this package first

```shell
$ brew update
$ brew install fvcproductions
```

> now install npm and bower packages

```shell
$ npm install
$ bower install
```
